# AI-Powered Multi-Language Translator 🌍

AI-Powered Multi-Language Translator is an NLP-based web application developed using Flask and Python that translates text between multiple languages using AI-powered translation techniques.

## Features

- Multi-language text translation
- NLP-based language processing
- Real-time translation
- Interactive and modern UI
- Fast and accurate translations

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Deep Translator
- NLP Techniques

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

## Open in Browser

```plaintext
http://127.0.0.1:5000
```
